# Getting Started

## Install

FinTrack is hosted on PyPI...

```console
$ pip install fintrack
```

## Basic Usage

FinTrack kan be used as a console/command line tool or as a Python module:

### Console

```console
% export DATE_ORDER=DMY
% export DATE_LANG=nl_NL
% LOG_LEVEL=DEBUG fintrack version

ðŸ¦ DEBUG fintrack.util - using date language nl_NL
ðŸ¦ DEBUG fintrack.tracker - using /Users/xtof/.fintrack
0.0.1

% date
Sun Jun  8 16:21:38 CEST 2025

% fintrack record 125 "starting balance" today
% fintrack table

| timestamp   |   amount | description      | uid                                  |

| vandaag     |      125 | starting balance | d1d477e7-9d7f-4047-b8d4-7c39386e7f9d |

% cat ~/.fintrack/records.json

    "amount": "125",
    "description": "starting balance",
    "timestamp": "2025-06-09T16:51:01.726441",
    "uid": "d1d477e7-9d7f-4047-b8d4-7c39386e7f9d"

] 
```

### Python Module

```python
>>> import json
>>> from fintrack.tracker import Tracker
>>> from fintrack.util import ClassEncoder
>>> tracker = Tracker()
>>> tracker.version()
'0.0.1'
>>> tracker.record(-200, "first payment")
>>> print(json.dumps(list(tracker), cls=ClassEncoder, indent=2))
[
  {
    "amount": "125",
    "description": "starting balance",
    "timestamp": "2025-06-09T16:51:01.726441",
    "uid": "d1d477e7-9d7f-4047-b8d4-7c39386e7f9d"
  },
  {
    "amount": "-200",
    "description": "first payment",
    "timestamp": "2025-06-09T16:53:09.782406",
    "uid": "51102227-472e-4443-a681-e18d597c6120"
  }
]
>>> print(tracker.table)
+-------------+----------+------------------+--------------------------------------+
| timestamp   |   amount | description      | uid                                  |
+=============+==========+==================+======================================+
| vandaag     |      125 | starting balance | d1d477e7-9d7f-4047-b8d4-7c39386e7f9d |
+-------------+----------+------------------+--------------------------------------+
| vandaag     |     -200 | first payment    | 51102227-472e-4443-a681-e18d597c6120 |
+-------------+----------+------------------+--------------------------------------+
```

## Planning

FinTrack also allows you to make plans using PlannedRecords:

```console

% fintrack plan -125 "groceries" "every friday", "groceries on {date}"
% fintrack plans table

| schedule         |   amount | description   | uids                | uid                                  |
```
